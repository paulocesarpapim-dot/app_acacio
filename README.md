**APP Acacio - Gestão de Produtos**

**About**

Esta é uma aplicação web para gerenciamento de produtos do Acacio.

**Tecnologias:**
- React 18
- Vite
- Tailwind CSS
- React Query
- React Hook Form
- Express API

**Prerequisites:**

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Make sure the backend API is running (api/server.js)

```bash
# Install dependencies
npm install

# Run the app in development
npm run dev

# Build for production
npm run build
```

**API Setup:**

Navigate to `api/` folder:
```bash
cd api
npm install
npm start
```

The API will be available at `http://localhost:3000`
